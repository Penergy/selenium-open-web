### 安装python3和virtualenv
1. 安装python3
   下载安装windows：https://www.python.org/
2. 配置python3，选择自定义安装
3. 安装virtualenv
    ```
    $ pip install virtualenv
    ```

### 启动该程序

---

1. 启动virualenv   
```
$  cd Scripts
$ .\activate
```
2. 打开Flask web服务
```
$ cd root
$ python index.1.py
```

### 打开flask应用
---
地址为：127.0.0.1:5000或者10.100.3.13:5000
